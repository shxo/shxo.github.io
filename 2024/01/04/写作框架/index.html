二、素材

  2. 随时记录

    • 记录你的碎片思考

四、框架
原则：
1、先框架再细节
2、一次只做一件事。一篇文章只写一个核心价值观

框架：
1、干货三件套：是什么+为什么+怎么做
2、观点三步法：总+分+总。步骤：
1、故事、案例、场景、痛点引入总论点
2、分论点从不同角度去论证
3、结尾提炼总结升华。强调、倡议、号召、口号，呼吁倡导读者怎么样
3、同维度并列拆分
4、逻辑递进拆分
5、不同维度罗列拆分
6、清单体简单罗列要点
7、同一点不同案例罗列法
8、人物故事框架：时间顺序法
1、通读人物材料，记录关键时间点和关键事件
2、根据第一步的记录，梳理一条人物经历的时间线
3、将整条时间线，按照关键事件，分成几个时间线
4、一个时间段写一个经历概括

故事框架：
1、故事三步法：冲突——行动——结局。变式：
1、问题——行动——转折
2、结果——原因——问题
3、结果——转折——原因
2、故事四步法：起承转合
3、故事七步法：目标——阻碍——行动——结果——意外——转折——结局

网文4种框架
1、观点+N个事例
1、第一步开头：热点开场+观点
2、第二步框架：具体案例3个
3、第三步结尾：强调观点
2、大观点+N个小观点
1、大观点要打破常规
2、每个小观点有一个案例支持
3、结尾辩证升华
3、观点+N种角度+金句重申观点引人深省
4、写人物：观点+人物的N个故事

八、文章逻辑清晰
技巧：

  1. 结论先行
    1、先说结果 再说原因
    2、先说目标 再说方法
    3、先说总体 再说展开
  2. 归纳分类
    标准：互相独立，完全穷尽
  3. 调整顺序
    逻辑就是信息的排序
    排序的方法：
    1、按重要程度顺序
    2、按因果关系顺序
    3、按结构关系顺序
    4、按时间顺序排序
    5、按推进步骤排序
  4. 删减信息
    减少逻辑干扰 // TODO 减法写作

十一、写文章，是先做加法，在做减法的过程
加法写作
1、不该省略的信息，要补上
2、已有的信息，该详细的要详细
3、增加论证纬度
4、增加举例数量

减法写作
1、聚焦主题，删掉偏离主题的内容
2、优化重复表达，没有增量不必增加。重复表达：
1、相近的观点
2、同质的案例
3、不同的说辞表达相同的意思
4、不言自明型重复
5、镜像重述：简单的用肯定和否定的形式表达同一个意思
3、从头朗读到尾，精简每一句话。如何删？
1、不必要的代称
2、删掉不必要的因果词
3、多删一点“的”、“是”、“了”
4、删掉累赘词
